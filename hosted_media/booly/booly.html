BOOLY (there are LOTS of bugs in this, heh)

. - Print bool to screen
# - Print next 8 bools as byte char to screen
^ - Get user input (anything non-0 is 1, cancel is 0) and store it in cell
< - Prev cell
! - Invert cell and go to next cell
[ - If cell = 1, do code within [], else go to ]
] - Ends a loop block
> - Next cell (would really be !<!, is just a shortcut)
