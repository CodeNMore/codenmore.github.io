Mash Race!

A Button Mashing Multiplayer 3D Racing Game by CodeNMore

Sorry, your browser must support the canvas element!

Instructions:
Green (player 1) presses the 'W' key repeatedly to boost forward.
Blue (player 2) presses the 'UP ARROW' repeatedly to boost forward.

Information:
Mash Race was made in (well under) 48 hours for the Mini Ludum Dare, number 54.
It is a button-mashing game, where 2 players press a button the fastest that they can to win a race!
The game was made using
- HTML / JavaScript / CSS
- Three.js (Open-Source 3D JavaScript Library)
- Brackets as a text editor / live debugger

The code is open-source, and can be found HERE!
